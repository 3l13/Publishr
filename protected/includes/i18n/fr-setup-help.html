Préfixe

Si vous avez un nombre limité de bases de données, vous pouvez utiliser un prefixe pour vos tables et ainsi partager une base de donnée.

Dépôt

Le dépôt sert à entreposer les fichiers téléchargés. Ils sont placés dans le sous-dossier correspondant à leur type : les images seront placées dans le sous-dossier images, tant dis que les fichiers mp3 seront placés dans le sous-dossier mp3.

N'oubliez pas de modifier les droits d'accés du dossier pour permettre l'enregistrement des fichiers.
